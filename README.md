# ALGO GROUP — Mind Map

Interactive logistics knowledge mind map for ALGO GROUP.

## Features
- 🗺️ Interactive mind map with zoom & pan
- ✏️ Add, edit, delete nodes
- 🎨 Custom node colors
- 💾 Auto-save to localStorage
- 📥 Export: PNG, PDF, JSON
- 📤 Import JSON
- 🌙 Dark/Light theme
- ⌨️ Keyboard shortcuts

## Shortcuts
- `Scroll` — Zoom in/out
- `Drag` — Pan the map
- `Click` — Select node
- `Double Click` — Edit node
- `N` — Add child node
- `Del` — Delete selected node
- `Esc` — Deselect

## Deploy

```bash
npm install
npm run dev
```

Deploy to Vercel by connecting this repo.
